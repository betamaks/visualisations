# Observability Steering

Observability visuals are hosted visuals first. Notion is optional and should not be assumed.

## Default deliverables

- One hosted visual under `observability/<topic>/<variant>/index.html`
- One topic manifest at `observability/<topic>/visual.meta.json`

## Visual rules

- Flexible sizing is allowed.
- Favor the visual language established by the AI agent architecture reference:
  - dark surfaces
  - restrained glow borders
  - mono labels paired with stronger display typography
  - system-diagram composition
  - compact explanatory copy
- Keep the page focused on the visual itself rather than article-like longform writing.

## Workflow rules

- Do not create a Notion article unless the user explicitly asks for one.
- Prefer reusable observability theme tokens and shared layout patterns over one-off page-local styling.
