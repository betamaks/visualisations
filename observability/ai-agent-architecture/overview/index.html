MCP Server Capabilities
MCP standardizes how the agent discovers and calls tools (JSON schemas + JSON-RPC). What each server does internally is entirely up to the vendor — from thin API wrappers (Atlassian) to multi-query orchestrators and AI-backed analysis (New Relic). The only universal layer: auth + API call + return structured JSON.
📊

New Relic MCP Server — 27 tools

  • execute_nrql_queryThin wrapper — runs NRQL via NerdGraph, returns raw JSON. Auth + call + return.
  • list_alert_policiesThin wrapper — fetches alert conditions and thresholds. Pure API passthrough.
  • analyze_transactionsOrchestrator — runs multiple NRQL queries, identifies slow/error-prone transactions, compiles report.
  • analyze_golden_metricsOrchestrator — fetches throughput/latency/error rate, applies health assessment heuristics.
  • natural_language_to_nrqlAI-backed — converts plain English to NRQL. Uses NR AI or embedded LLM call.
  • generate_alert_insightsAI-backed — root cause analysis, impact assessment, recommended actions for an issue.

Agent Rulesets
User-defined configuration injected into every LLM prompt. These shape agent behavior — response format, investigation protocols, and safety constraints. Click to expand.

Response Style

Enforces ~5 line outputs, forbidden phrases, one-word answers when possible. No AI boilerplate.

# Example:
User: Is 11 prime?
Agent: Yes
# Forbidden: "Based on the information..."

NR Investigation Protocol

Maps accounts (1111111 dev, 2222222 live). Time windows start narrow, expand. Cross-table correlation keys.

# Time window strategy:
1. SINCE 3 hours ago
2. SINCE 1 day ago (if empty)
3. SINCE 7 days ago (simple only)

Truth & Verification

Labels claims as [Inference], [Speculation], [Unverified]. Fetch before reasoning about recent tech.

IF date 2025+ AND recent tech → FETCH first
IF unverified → label it
# Never present guesses as fact

Timestamp & Epoch

Default UTC+2. Converts with ms precision, round-trip validates. Critical for time-bounded NR queries.

1. Capture TZ Europe/Tallinn
2. Convert to epoch ms
3. Round-trip verify
4. Use in NRQL SINCE/UNTIL

Security & Privacy

Never expose secrets. Env vars for config. Never commit credentials. Input validation mandatory.

✅ Environment variables for secrets
✅ HTTPS/TLS only
❌ Never hardcode API keys
❌ Never log sensitive user data

MCP Interaction Rules

Read tool schema before calling. Correct NR account per env. Handle NRDB:1109 via chunking. Max LIMIT 5000.

1. Read tool JSON descriptor
2. Validate required params
3. CallMcpTool(server, tool, args)
4. Handle errors per ruleset
# NRDB:1109 → chunk into 7d windows
