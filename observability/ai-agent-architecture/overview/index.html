MCP Server Capabilities
Five MCP servers provide tool access via JSON-RPC. NerdGraph is accessed directly via instruction-driven GraphQL when MCP queries hit complexity limits or timeout.

Agent Rulesets & Skills
Rulesets are always-injected configuration. Skills are on-demand investigative capabilities invoked by the agent when needed (e.g., post-incident RCA, instruction improvement retrospectives).

Response Style

Enforces ~5 line outputs, forbidden phrases, one-word answers when possible. No AI boilerplate.

# Example:
User: Is 11 prime?
Agent: Yes
# Forbidden: "Based on the information..."

NR Investigation Protocol

Maps accounts (dev, live). Time windows start narrow, expand. Cross-table correlation.

# Time window strategy:
1. SINCE 3 hours ago
2. SINCE 1 day ago (if empty)
3. SINCE 7 days ago (simple only)

NerdGraph Fallback

When MCP times out or hits complexity limits, agent constructs raw GraphQL via NerdGraph async polling (60-70s).

IF MCP timeout OR NRDB:1109
THEN NerdGraph async query
# 60-70s polling, chunk if needed

Truth & Verification

Labels claims as [Inference], [Speculation], [Unverified]. Fetch before reasoning about recent tech.

IF date 2025+ AND recent tech → FETCH first
IF unverified → label it
# Never present guesses as fact

Skill: RCA Writer

Post-incident RCA document generation. Drives data collection, timeline, impact quantification, and root cause analysis.

INVOKE WHEN: investigation complete
Produces reviewer-ready RCA
# Structured, evidence-based

Skill: NR Instruction Review

Retrospective after investigations. Reviews output quality, identifies instruction gaps, proposes rule changes.

INVOKE WHEN: post-investigation
