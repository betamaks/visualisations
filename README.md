# Visualisations

Hosted interactive visuals for embedding into Notion.

## Publishing

- GitHub Pages base URL: `https://betamaks.github.io/visualisations/`

## Structure

- `veterinary/<topic-slug>/<variant>/index.html`

Example:

- `veterinary/dachshund-heart-valves/cycle/index.html`
- `veterinary/dachshund-heart-valves/maximal/index.html`

## Visual rules

- Build embed-first pages, not full articles.
- Keep desktop visuals within one bounded canvas.
- Avoid long content below the fold.
- Keep detailed explanation in Notion, not inside the hosted page.
- Keep hosted URLs stable so existing Notion embeds continue to update.
